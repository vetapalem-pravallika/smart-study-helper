## ⚙️ How to Install

1. Download or clone the repository.  
2. Open Chrome → `chrome://extensions/`  
3. Enable **Developer mode**.  
4. Click **Load unpacked** → select the project folder.  
5. Click the extension icon in the toolbar to use it.
